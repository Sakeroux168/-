# 🚀 视频运营全工作量工具 (Video Operation Tools)

这是一个专为视频运营人员打造的自动化工作流工具集。主要涵盖了**B站视频链接批量提取、官方数据接口抓取、本地视频智能匹配打标签、以及多文件夹视频同步重命名**等极其解放双手的功能。

---

## ⚙️ 核心工作流 (Workflow)
本仓库中的前三个工具可以组合成一个完美的自动化流水线：
1. **获取链接**：用 `魔法书签` 在网页一键提取指定日期的B站视频链接。
2. **抓取数据**：用 `爬取标签.py` 自动去B站后台获取这些视频的准确标题和热门标签。
3. **给视频贴标**：用 `贴标签.py` 自动比对本地视频库，把抓下来的标签直接写入到视频的文件名里。

---

## 📂 详细工具说明

### 1. 🪄 魔法书签，爬取视频链接.txt (浏览器脚本)
* **功能**：一个非常轻量的 JavaScript 浏览器书签代码。
* **使用场景**：在B站的视频列表页运行，可以一键筛选出**指定日期范围**内的所有视频，并提取它们的真实链接。
* **输出结果**：不仅会自动把所有链接复制到剪贴板，还会自动下载一个名为 `bilibili_links.txt` 的文件，供下一个脚本使用。
* **使用方式**：在浏览器随便开一个网页，然后收藏，在右上角的收藏夹里面找到你刚才收藏的网页，右键它点击编辑，把代码贴进去后就可以使用了
* **注意事项**：输入年月日的时候，以从旧到新的规则输入  比如2025-01-01 到 2026-01-01

### 2. 🕷️ 通过链接爬取视频名字标签.py
* **功能**：自动读取上一步生成的 `bilibili_links.txt`，调用B站官方的 API 接口。
* **原理解析**：它可以智能解析各种长短链接提取真实的视频信息，并自动请求 B站服务器，精准抓取视频的“完整标题”和带 `#` 号排版好的“视频标签”。
* **输出结果**：生成一份排版精美的 `批量标签结果.txt`。内置了礼貌延时（防封IP）机制。

### 3. 🏷️ 贴标签.py
* **功能**：AI 智能视频文件名标签追加工具。
* **原理解析**：读取 `批量标签结果.txt` 里的数据，然后让你输入本地视频所在的文件夹。脚本会通过 **精确包含匹配** 和 **模糊相似度算法 (difflib)**，自动找出哪个本地视频对应哪条标签数据。
* **防呆设计**：自动过滤 Windows 系统不支持的文件名特殊字符（如 `\/:*?"<>|`）；如果视频名字里已经有 `#` 会自动跳过防重复；执行前会列出详细的**修改预览**让你确认，极其安全。

### 4. 🔄 批量替换标题2.0.py
* **功能**：剪辑前后的多文件夹标题同步神器。
* **使用场景**：比如 A 文件夹里是原视频（名字很长），B 文件夹里是刚剪辑好的视频（名字只剩序号）。
* **原理解析**：脚本会自动提取 B 文件夹视频名称的“后缀数字”，并与 A 文件夹视频名称的“前缀数字”进行强绑定配对，让剪辑后的视频一键恢复成原本复杂的标题。支持智能排除特定子文件夹。
